# 家常菜

## 炒菜

### 腊肉系列

> 腊肉小炒

#### 腊肉炒泥蒿

主料：腊肉、泥蒿

辅料：蒜末

> 步骤

1. 准备泥蒿一斤左右，腊肉50-100g，大蒜4瓣；

2. 掐去泥蒿叶，及泥蒿较老的部分茎，择成小段；

3. 将腊肉切小块焯水。焯水时间视情况而定，一般越大块焯水时间越长，一般至少一二十分钟，炒出腊肉中的盐分，但要注意不要将腊肉的味道全部煮没了。也可以提前将腊肉泡水去除一部分盐分；

4. 将腊肉切小片，最好肥瘦分开，大蒜切碎；

5. 热锅倒油，下腊肉中肥的部分，爆炒出油；

6. 大火，下蒜末， 泥蒿，爆炒一至两分钟，加适量的盐、味精、十三香、生抽等，继续翻炒均匀；

7. 下腊肉瘦肉部分，继续翻炒；

8. 出锅.

> 注意事项

1. 泥蒿表面光滑，不容易沾盐，腊肉表面粗糙更容易沾盐，尽量先炒泥蒿时加盐，如果一起放在锅里，盐会主要沾在腊肉上，使得腊肉更咸;

2. 炒泥蒿标准：大火翻炒一两分钟至泥蒿变色（颜色更深）（断生），即可加调料，炒的太老了，泥蒿会变得软趴趴;

3. 腊肉如果肥瘦不分的话，可以先炒出腊肉中的油，盛出来后再炒泥蒿，加调料炒熟之后再倒回去；

4. 家庭燃气灶火力较小，炒菜时尽量在锅中铺平，加热一会再翻炒.
